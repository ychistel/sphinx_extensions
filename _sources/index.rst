Extensions pour Sphinx
========================

.. _sphinx-contrib: https://github.com/sphinx-contrib

Les extensions pour sphinx permettent d'ajouter des fonctionnalités qui améliorent la création de contenus.

Il exite déjà des extensions que l'on peut trouver sur le site sphinx-contrib_.

Pour mon usage personnel, j'ai développé les extensions suivantes:

-   La numérotation automatique des exercices dans un projet sphinx.
-   L'insertion d'un éditeur de code Python avec PyScript et Pyodide.
